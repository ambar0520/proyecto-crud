#Proyecto CRUD con Flask, MYSQL Y JavaScript

Este proyecto es una aplicación web que implementa operaciones CRUD (Create, Read, Update, Delete) utilizando Flask como framework backend en Python y MySQL como base de datos. El frontend está desarrollado con HTML, CSS y JavaScript, incluyendo validaciones en los formularios.

##Funcionalidades  principales

Registro de usuarios con almacenamiento en MySQL.

Inicio de sesión con validación de credenciales.

Visualización de usuarios en una tabla (ID, nombre, contraseña).

Actualización de contraseña por usuario.

Eliminación de usuarios por ID (para evitar borrar duplicados).

Validaciones en frontend con JavaScript (campos requeridos, alertas).

Estilos básicos en CSS para formularios y tablas.
